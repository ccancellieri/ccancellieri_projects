# Keystone — Unified Identity and Access Management Platform

**Organization:** FAO (Food and Agriculture Organization of the United Nations)
**Author:** Carlo Cancellieri (Lead Software Engineer, architect)
**Period:** 2020–present
**Status:** Production — serving 25,000+ users across 50+ countries

## Overview

Keystone is FAO's unified Identity and Access Management (IAM) platform, designed to provide centralized authentication and fine-grained authorization across the Organization's digital ecosystem under the Hand-in-Hand Initiative.

## Key Features

- **RBAC/ReBAC authorization model** — Role-Based and Relationship-Based Access Control
- **25,000+ users** across 50+ member countries
- **Cross-divisional integration** — Statistics (ESS), IT (CIO), regional offices
- **Cloud-native** — GCP, Kubernetes, Terraform
- **Standards-based** — OAuth 2.0, OpenID Connect
